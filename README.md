## 📌 Sobre o Aplicativo

O **Cardápio Virtual** é um aplicativo mobile desenvolvido em Flutter que simula um sistema de pedidos online para restaurantes.
O usuário pode realizar login, visualizar produtos por categorias, pesquisar itens, adicionar produtos ao carrinho e finalizar pedidos.

## ▶️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/VictorHSSouza/app_cardapio_virtual.git
```

### 2. Acesse a pasta do projeto

```bash
cd app_cardapio_virtual
```

### 3. Instale as dependências

```bash
flutter pub get
```

### 4. Execute o aplicativo

```bash
flutter run
```

## 📱 Fluxo das Telas

### 🔐 Tela de Login

Responsável pela autenticação do usuário no aplicativo.

### 📝 Tela de Registro

Permite que novos usuários realizem cadastro na plataforma.

### 🏠 Tela Inicial (Home)

Exibe o cardápio virtual com categorias, busca de produtos e acesso ao carrinho.

### 🛒 Tela de Carrinho

Mostra os itens adicionados ao pedido, quantidade, subtotal e finalização da compra.

### 👤 Modal de Perfil

Apresenta informações do usuário logado e opção de logout.

### ➕ Modal de Quantidade

Permite selecionar a quantidade de produtos antes de adicionar ao carrinho.
